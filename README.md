# HTML & CSS Portfolio

A simple personal portfolio website built with HTML and CSS.

## Features
- Hero/intro section
- Sidebar navigation
- About, Skills, Education, Experience, and Contact sections
- Responsive layout (CSS-only)

## Tech Stack
- HTML5
- CSS3

## Project Structure
- `index.html` – main portfolio page
- `globals.css` – base/global styles
- `css/` – modular CSS files for different sections and utilities
- `images/` – images and assets

## Getting Started
1. Clone or download this repository.
2. Open `index.html` in your browser (double-click or drag into a browser window).

No build tools or dependencies are required.

## Customization
- Update the content in `index.html` with your own details.
- Adjust colors, fonts, and layout in the CSS files (e.g., `css/globals.css`, `css/hero.css`, `css/sidebar.css`).
